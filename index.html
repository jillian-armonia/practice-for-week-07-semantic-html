My HTML Cheat Sheet

Headings

Headings with tags from 'h1' to 'h6' are used for titles, headings, and subheadings. By using the appropriate tags, you can optimize the searchability of your website. You might be tempted to use h1 because it draws attention, but it should only be used for the title. Headings can help identify blocks and sections.

Paragraphs

Paragraphs are also block elements like headings, but they contain more long-form content. They have the 'p' tags.

Lists

Unordered Lists

Unordered lists have the 'ul' tags. Each list item has a bullet point. Each list item goes under the 'li' tag.

Types of lists:

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists have the 'ol' tags. Each list item will be numbered, starting from 1. Each list item also goes under the 'li' tag.

Anchors (i.e. links)

Anchor element has an 'a' tag. It is used to put links to a different webpage in your website. Go to DuckDuckGo to search for more information.

Setting your target attribute to "_blank" will open a new window when you click the link. See MDN for more details.

Images

You can use the img tag to add images to your page.

Tables

Tables are a great way to organize and show data on your webpage. Start with a 'table' tag. Then inside the 'table' tag, use the 'thead' tag to create a header row for your table. Use the 'tbody' tag for the main content of your table. Lastly, use the 'tfoot' tag for the footer row of your table.

Articles

The article element is a self-contained composition that can easily be distributed or copied. This can be forum posts, widgets, etc.

Footer

The footer element is always at the end of the page. It usually contains links for navigation or other webpages for more information.

Header

Not to be confused with headings, the header is a section that introduces the page and usually has the navigation links here. It might contain the logo and information about the author.

Nav

The nav element, or navigation element, contains links to all the other webpages in your website.

How to Make a Form

Tips on Designing Your Form

  • Make it simple and focused.
  • Avoid spacing your form and making it big.
  • Listen to a screen reader to check how the form structure sounds like.

Form Elements

Form

You can use the 'form' tag. It is like the container element and configures how this section behaves. The action attribute is the link to where the data collected is sent to. The method attribute is the link what HTTP method is gonna be used.

Label

The label element helps with giving names and levels to each input field in the form. You can use the 'label' tag with the for attribute for the corresponding input.

You can wrap a 'p' tag around the 'label' and 'input' tags to help with styling it.

Input

This is a self-closing tag. Using the type attribute, you can set the what type of input you need from the client. If you use email, it will only accept email addresses from the client.

Text Area

This is not a self-closing tag like input. It's a long form text input. You can use this for long comments.

Button

Clicking this element, you can submit your form.

Structural Element for Forms

Fieldsets

This acts like a section inside your form element. It helps organize a set of choices, especially when you are putting radio-type input.

Legends

This is a special label for the fieldset. It acts as fieldset header.
